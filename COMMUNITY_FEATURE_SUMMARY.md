## 🚀 下一步配置

### 1. **配置Supabase**
1. 复制 `.env.example` 为 `.env.local`
2. 填写你的Supabase项目信息：
   ```
   NEXT_PUBLIC_SUPABASE_URL=你的项目URL
   NEXT_PUBLIC_SUPABASE_ANON_KEY=你的匿名密钥
   ```

### 2. **创建数据库表**
在Supabase控制台执行提供的SQL语句，创建`profiles`和`posts`表。

### 3. **启动应用**
```bash
npm run dev
```

### 4. **访问论坛**
- 主页点击"社区论坛"进入
- 或直接访问 `/community`
